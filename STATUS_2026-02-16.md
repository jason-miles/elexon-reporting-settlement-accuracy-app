## Useful commands

```bash
# Sync app to Databricks
cd /Users/jason.miles/vibe-coding-repos/elexon-reporting-settlement-accuracy-app2/frontend
databricks sync . /Workspace/Users/jason.miles@databricks.com/elexon-insights-detection-app --profile elexon

# Deploy app
databricks apps deploy elexon-insights-detection-app --source-code-path /Workspace/Users/jason.miles@databricks.com/elexon-insights-detection-app --profile elexon

# Deploy DABS
cd /Users/jason.miles/vibe-coding-repos/elexon-reporting-settlement-accuracy-app2
databricks bundle deploy -t dev --profile elexon

# Run app locally
cd frontend && npm install && npm run build && npm run preview

# Push to GitHub (after committing)
git push origin main
```
